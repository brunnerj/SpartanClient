LabVIEW Timing Toolkit

You can use the Timer VIs to keep track of the time of running processes.  You can start, stop, and reset a timer just as you could a stopwatch.  You can get the status of a timer to tell if it is running and what its time value is.  You can also synchronize events to occur at specific timer values.

The Timer VIs differ from the Timing VIs in the Time & Dialogue functions pallet which use the millisecond timer on your computer.  Timing Toolkit VIs allow you to create your own timers and to use them in more powerful ways.

 You can access the timer VIs by selecting Functions»Timer.

The timer VIs use the Timer RefNum control from the Controls»Timer palette.

The Timer RefNum can be used with the following VIs.

Create Timer

Destroy Timer

Get Timer Status

Not A Timer

Reset Timer

Start Timer

Stop Timer

Wait on Timer
